## Meeting Notes

### This Week in HLSL

19 PRs were merged this week across a few areas:

- **HLSL Frontend**
  - Fixed miscompile from emitting `lifetime.start` after copy-in for `inout`
    parameters.
  - Corrected SPIRV CHECKs in derivative tests.
  - Added codegen tests for `ConstantBuffer<T>` element access and
    initialization.
- **DirectX Backend**
  - Fixed missing `llvm.dx.resource.getbasepointer` handling.
- **Debug Info & PDB**
  - Added DXContainer support and `--dxcontainer` to `llvm-pdbutil`.
- **Offload Test Suite**
  - New tests: RWByteAddressBuffer 16-bit clobbering check, `isnan()` coverage,
    hull/domain system-value tests, and a mesh shader line-geometry test.
  - More progress on moving logic out of each target into the generic
    abstraction layer.
  - The split-build feature is live, which should improve build throughput.
- **SPIR-V Backend**
  - Legalization and lowering fixes including G_PHI legalization and lowering
    of nested undef in constant aggregates.
  - Fixes for verifier/crash cases on indirect calls with aggregate arguments.

For a full list of merged PRs, see the appendix.

There are also 16 newly opened PRs, but 53 currently ongoing PRs which is up
from last week.
