## Meeting Notes

### This Week in HLSL

This was a slower week in HLSL, but we did see 27 PRs merged:

- **HLSL Frontend**
  - Diagnostic improvements for for entry functions and resource parameters
  - RWTexture2D was implemented and the tests for textures were cleaned up
- **DirectX Backend**
  - Some container related flags for debug and PDB handling were implemented
- **SPIR-V Backend**
  - Multiple improvements to lowering and legalization.
  - Various refactorings
- **Offload Test Suite**
  - Various XFAILs were updated per implementation effort elsewhere
  - Infrastructure to run CI using lavapipe and WARP was improved

For a full list of merged PRs, see the appendix.
