## This week in HLSL

**HLSL (clang)**
- Codegen for global resource array initialization (llvm/llvm-project#198891, hekota)
- Matrix support for `splitdouble` (llvm/llvm-project#200257, joaosaffran)
- Matrix-layout-keyword codegen (llvm/llvm-project#198887) with matching offload tests (llvm/offload-test-suite#1211) (farzonl)
- constexpr vector element conversions (llvm/llvm-project#195173, llvm-beanz)

**DirectX**
- DXContainer parsing improvements: magic check (llvm/llvm-project#198586) and null-buffer UB fix (llvm/llvm-project#200865) (kuilpd)
- llvm-objcopy ILDB header stripping (llvm/llvm-project#200906, kuilpd)
- DXIL lowering for SampleBias (llvm/llvm-project#199745) and SampleGrad (llvm/llvm-project#200871) (Icohedron)
- IR printing moved into DXILPrettyPrinter (llvm/llvm-project#198318), debug labels dropped (llvm/llvm-project#197490), DCE/DSE disabled at -O0 (llvm/llvm-project#192520) (hvdijk)
- llc now generates shader debug file name (llvm/llvm-project#199555, dzhidzhoev)
- Obsolete cbuffer layout test removed (llvm/llvm-project#200307, hekota)

**SPIR-V**
- Various general SPIR-V fixes: int-to-int convert opcode signedness (llvm/llvm-project#200791, llvm/llvm-project#201116), Sext decoration for `signext` params (llvm/llvm-project#200801), s128 in legalizer (llvm/llvm-project#199998), reject module-level inline asm (llvm/llvm-project#199992), out-of-bounds diagnostics on function-type metadata (llvm/llvm-project#200601), i8 pointer fix for NULL events in OpEnqueueKernel (llvm/llvm-project#201125), service-function fix for declaration-only modules (llvm/llvm-project#199964) (aobolensk)
- `reqd_sub_group_size` recognized for SubgroupSize (llvm/llvm-project#199521, svenvh)
- SPV_KHR_poison_freeze implemented (llvm/llvm-project#198037, MrSidims)
- Enqueue-empty-kernel fix (llvm/llvm-project#187671, idubinov)

**offload-test-suite**

- InterlockedOr test added (llvm/offload-test-suite#1195), runner-status script (llvm/offload-test-suite#1185), runner hardware docs updated (llvm/offload-test-suite#1248) (bob80905)
