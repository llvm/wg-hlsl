## Meeting Notes

### This Week in HLSL

65 PRs were merged this week across a few areas:

- **HLSL Frontend**
  - Improved handling of CBuffers in arguments and globals.
  - User defined types no longer have implicit constructors.
  - An issue with transpose not respecting matrix layout was fixed.
  - Infrastructure for differentiating resources was improved.
  - The HLSL function QuadReadAcrossDiagonal was implemented.
  - The -fspv-preserve-interface compatibility flag was implemented.
- **DirectX Backend**
  - We now clean up unused resource globals before assigning bindings.
  - SimplifyCFG no longer generates invalid `i8` lookup tables.
- **Debug Info & PDB**:
  - We can now create PDBs alongside DirectX objects.
  - There has been a lot of progress on translating modern LLVM's debug info to
    DXC / LLVM 3.7 debug info.
- **Offload Test Suite**:
  - More raytracing tests have gone in
  - The work to generalize the three APIs and reduce code duplication is making
    progress.
  - More tests were added
  - Various infrastructural fixes and improvements were put in place for CI.
- **SPIR-V Backend**
  - A number of lowering improvements went in
  - Some verifier crashes were fixed

For a full list of merged PRs, see the appendix.

There are also 22 newly opened PRs and 47 currently ongoing PRs.
